# 🌍 AI Smart Travel Planner

AI Smart Travel Planner is an AI-powered web application that helps users generate personalized travel itineraries based on their destination, budget, trip duration, and travel preferences. The application uses Google's Gemini AI to provide intelligent travel recommendations and detailed day-wise plans.

## ✨ Features

*  AI-generated travel itineraries
*  Budget-based trip planning
*  Day-wise travel schedule
*  Local food recommendations
*  Travel tips and packing suggestions
*  Interactive and modern user interface
*  Travel-themed landing page with video background

## Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/SnehaKarmakar1505/AI-Smart-Travel-Planner.git
```

### 2. Navigate to the Project Directory

```bash
cd AI-Smart-Travel-Planner
```

### 3. Install Required Packages

```bash
pip install flask google-genai
```

### 4. Configure Gemini API Key

Add your Gemini API key inside the application configuration.

### 5. Run the Application

```bash
python app.py
```

### 6. Open in Browser

```text
http://127.0.0.1:5000
```

## How It Works

1. Open the Smart Travel Planner website.
2. Explore the landing page and project overview.
3. Click on **Let's Explore**.
4. Enter:

   * Destination
   * Budget
   * Number of Days
   * Travel Type
5. Generate a personalized travel itinerary using Gemini AI.
